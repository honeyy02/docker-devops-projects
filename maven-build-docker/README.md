# Maven Build in Docker

## 📌 Overview
This project demonstrates building a Java application using Maven inside a Docker container.

---

## 🛠️ Tech Stack
- Docker
- Maven
- Java

---

## 🚀 Workflow
1. Copy source code into container
2. Run Maven build
3. Generate JAR file inside container
