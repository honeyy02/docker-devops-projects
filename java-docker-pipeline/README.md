# Java Docker CI/CD Pipeline 🚀

## 📌 Overview
This project demonstrates building, running, and pushing a Docker image for a Java application using Jenkins CI/CD pipeline.

---

## 🚀 Workflow
1. Checkout source code
2. Build Docker image
3. Run container
4. Push image to Docker Hub

---

## 🛠️ Tech Stack
- Docker
- Jenkins
- Java
